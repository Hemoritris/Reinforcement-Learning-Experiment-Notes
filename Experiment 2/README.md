# Experiment 2: MDP Modeling Analysis of CartPole-v1 Game

This experiment revolves around the classic OpenAI Gym / Gymnasium reinforcement learning environment, CartPole-v1. It completes the full modeling of the Markov Decision Process (MDP) 5-tuple, verifies the basic interaction logic of the environment, and aims to master the standardized MDP modeling methods for foundational RL environments.

## 📝 Experiment Overview
1. **Environment Comprehension**: Get familiar with the CartPole-v1 pole-balancing task, clarifying the environmental control objectives, the physical meanings of states, the action set, and the termination conditions.
2. **MDP Modeling**: Complete the definition and analysis of the state space, action space, state transition probabilities, reward function, and discount factor.
3. **Logic Verification**: Implement environment initialization, random action interaction, and visual execution using Python to verify the correctness of the modeling.

## 💻 Code
- Contains the Python implementation code for CartPole-v1 environment testing and visual interaction, which is used to verify the MDP modeling and fundamental environment interaction workflow.
